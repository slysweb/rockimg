# Rockimg.com

**Rockimg.com** is a browser-based image and GIF workshop: memes, fake-style screenshots, conversions, and dozens of small tools that run locally in your device when possible.

Tutorials and static posts live on the [blog](https://www.rockimg.com/blog.html). The same HTML is also mirrored on GitHub for sharing and discoverability (see [Open blog on GitHub](#open-blog-on-github) below).

## More on the product

- **Stack (high level):** Vue.js, Element Plus; Node.js and PHP (ThinkPHP) where applicable; image work via Sharp, FFmpeg.wasm, and optional in-browser models.
- **How to use:** open [Rockimg.com](https://www.rockimg.com/), pick a tool (the blog table on this page links straight into each one), work in the browser, then download.
